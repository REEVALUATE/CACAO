## About CACAO

The Cultural Artefact Contextual Ontology (CACAO) provides a semantic framework for modeling and contextualizing cultural heritage artifacts. CACAO integrates multiple well-established ontologies:

- **CIDOC-CRM 7.1.3** - Cultural heritage documentation
- **PROV-O** - Provenance tracking
- **FOAF** - Agent representation
- **Schema.org** - Web interoperability
- **ODRL** - Rights management

CACAO is part of the REEVALUATE project and follows OBO Foundry best practices.
